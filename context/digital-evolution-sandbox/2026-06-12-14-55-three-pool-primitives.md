### 本轮过程

### 设计宪法 (6条, 所有 agent 戴着干活)

1. 表型=序列固定函数(③): 基元只定义动作/编辑, 效果靠"动作撞有限世界"涌现, 严禁附带表型增益或手写"谁强"。
2. a/φ 干净、严禁夹私货: 不准隐藏优先级/权重/加成; 目标必须是确定的序列编辑或确定的猎物匹配。
3. 已锁机制不可破: Z公式 kills=min(b,a·z)+自损kills/z; F三吐 f·波及格·p_leave; P·μ·rate×spectrum·耦合繁衍。
4. 红皇后对称: 无"绝对强", 只有"比此刻这群对手强"; 必须能造权衡谱 / 循环克制。
5. 抗体生物对齐: SHM / CDR loop / 亲和力成熟 / 表位竞争 / 免疫逃逸。每条标生物对应。
6. 可玩性: 无退化主导策略。

### 突变池 v1 (策划 agent ac8097)

**三大待拍点的推荐:**

- **A. SPECTRUM 形状 → 推荐乙(带概率候选列表), 但加硬规定: q 只能由序列算出, 不许手填常数。** 理由: 甲(单靶点)表达不了 SHM 的 AID 热点方向偏好(WRCY/RGYW), 且"想变多种靠插多 P"会致基元爆炸(一个真实热点要 P_K/P_R/P_H 三插槽); 丙(等概率)生物上错(转换>颠换, 物化近邻概率高), 且使 spectrum 层对 p(x) 透明。乙若规定 q_i(x) 由 x 局部序列特征算出(归一化计数 / BLOSUM 查表)、运行中不可调参、只读序列不读 count/对手/步数 → 同 P 插不同位 q 不同(上位效应从 spectrum 也出)、仍是 x 固定函数不破③、归 a 干净。
- **B. 编辑粒度 → 骨干 residue 点替换(对齐 CR9114 组合点突变 DMS) + 附加 motif 整体替换(大跳变穿越 fitness valley); 先不做插槽洗牌(与骨架调制上位效应缠绕, backbone 未标定前不可控)、坚决不做 indel(动插槽数破"骨架定死", 4 谱系边界糊掉)。**
- **C. backbone 不动 → 强同意。** 三层理由: ①科研控制变量(骨架=恒量, 插槽=变量, 骨架可漂则 a/φ 边界糊); ②物种隔离是频率依赖的必要条件(4骨架=4生态位, 可漂则塌成灰色地带); ③实现可控(骨架不动→搜索空间=插槽内容笛卡尔积, 有限可枚举)。补充: 骨架本身是静默物种标签, 对抗基元可设计读骨架脚手架位 → 天然相克从骨架差异涌现(免费设计空间)。

**突变池基元清单 (v1):**

Residue 层(点替换):
- **AID-core / P_aic**: 将插槽某残基替换为候选(候选表=当前残基的4个物化近邻); p(x)=μ+α·(WRC-like三联体出现次数归一化), q_i=物化距离倒数归一化。生物=AID 识别 WRCY/RGYW 热点, 亲和力成熟主力。
- **Error-prone / P_ep**: 随机选插槽残基替换为任意合法型; p(x)=μ·β (β∈(1,4), 由错配标记残基计数定), q_i=均匀宽谱无向。生物=低保真聚合酶/error-prone PCR。
- **Hotspot-bias / P_hsb**: 专打某类功能残基(如带电荷)位点; 候选=同电荷族2-3个; p(x)=μ+γ·(目标位点存在flag), q_i 按 BLOSUM 列归一化。生物=CDR 带电残基热点突变(H-CDR3 的 R/K 偏好)。

Motif 层(整体替换):
- **Loop-swap / P_ls**: 整插槽 motif 换成同池另一合法 motif; p(x)=μ+δ·(该 motif 已出现超 n 代标记), q_i 按目标与当前编辑距离倒数。生物=CDR loop 整体构型切换(后期 loop 重塑大跳跃)。
- **Scaffold-test / P_sct**: 把 backbone 插槽 motif 替换为 null/截短变体(删功能不删位点); p(x)=μ 无加码(纯基线漂驱动)。生物=假基因化/功能丧失中性漂变。
- **Cross-pool-swap / P_cps**: 繁衍插槽 motif 换成突变池 motif(或反向); p(x)=μ+ε·(某池 motif 数>阈值θ), q_i 均匀。生物=基因功能转变(exon shuffling)。

**设计原则**: 三组对称覆盖 — P_aic+P_ls=有偏定向(AID热点); P_ep+P_sct=宽谱无向(低保真+中性漂); P_hsb+P_cps=特异约束(电荷偏好+功能转变)。各组频率依赖行为不同(定向→维持谱系/宽谱→四散探索多无功能/特异→构型跳变穿谷)。

**突变池待拍/风险:**
- q_i(x) 允许用 BLOSUM 归一化查表吗? BLOSUM 是公开标准矩阵但属生物统计先验 — 是否算"确定序列编辑非私货"(宪法2)? **最需用户确认的边界。**
- P_cps 候选表是运行时动态(已登记 motif)还是启动前静态锁定? 倾向静态(动态会让 a 随运行状态变)。
- **跨池耦合(突变池自报)**: ①突变耦合繁衍 → 繁衍强的株突变也多(F4周期短f大→更多拷贝→更多突变机会), 分析侧要记"突变来源的繁衍事件数"以分离 f 贡献; ②P_ls/P_cps 替换掉某 motif → 目标株可能从对抗基元猎物清单上消失(逃逸突变, 频率依赖来源), 实现侧须确认对抗结算用当前步序列快照非缓存。
- **顺带建议(非本池职责)**: 折叠 WHAT 推荐"加 z + 加抗被猎", 明确不推荐加 f(折叠加繁衍会糊 f 来源); step loop 推荐 对抗→繁衍(含突变)→K墙仲裁。

### 繁衍池 v1 (策划 agent acbb23)

Residue 层(单字母粒度):
- **R-F1 脉冲**: f=0.6, 上下左右4格, p_leave=0.3, 每4步。高频中强度脉冲撒种+中等迁出。涌现=稀疏区快填/满格蒸发/源格被自己稀释。生物=B细胞生发中心快速克隆扩增。
- **R-F2 囤积**: f=0.15, 原地, p_leave=0, 每3步。极低比例高频不迁出。涌现=源格count缓慢垫高抢更多比例空位/满格冻结/对抗杀出空位时最先受益。生物=B细胞留守生发中心。
- **R-F3 漂移**: f=0.35, 随机1格(8邻随机), p_leave=0.5, 每6步。涌现=地图孤立飞地/源格减少/飞地无后援易被清/落空旷区成前哨。生物=记忆B细胞外周血随机归巢。
- **R-F4 锚定**: f=0.08, 原地, p_leave=0, 每2步。超高频超低量原地堆叠。涌现=满格浪费/"快满差几个"极高效/依赖对抗先打空位/单独几无扩张。生物=长寿浆细胞骨髓龛位自我更新。

Motif 层:
- **M-FA 梯度波**: 向8邻中count最低2格各撒0.45, p_leave=0.2, 每8步。主动找空。涌现=快占低密度格/满格不浪费/填洼地/全满冻结。生物=趋化因子迁移(CXCL13/CXCR5)。**注:读邻格count是表型参数计算非手写φ,胜负仍走等比例仲裁。**
- **M-FB 爆发-静息**: 爆发相f=0.8(1次)→强制静息12步, 4格, 爆发p_leave=0.6, 每12步爆发。涌现=爆发瞬间向四邻各投0.8a/与对抗清场对齐则极有效/静息期被压制/节奏对齐错位产生时间窗博弈。生物=B细胞克隆爆发后分化沉默期。
- **M-FC 单向锋线**: f=0.55, 固定方向3格(正前1+斜前2, 序列编码), p_leave=0.25, 每5步。楔形推进。涌现=方向性锋线/背部裸露易被侧后攻/方向相对正面撞/方向不同可包抄。生物=淋巴细胞定向迁移(暗区→亮区)。

**权衡谱**: 快扩张易破端=R-F1+M-FB(撒种快但p_leave高致源格count低→被高z一次清); 慢站稳端=R-F2+R-F4(源格count高→对抗a大能扛但扩张近零)。例A激进锋线=M-FC+R-F1+M-FB; 例B据守反推=R-F4+R-F2+M-FA。

**设计原则**: f拉开(0.08~0.8)避免等比例仲裁失区分力; 时钟2-12步跨度让节奏博弈有内容; p_leave 0~0.6保证攻守真权衡; 能用residue不升motif。

**繁衍池待拍/风险:**
- **风险1**: M-FA"感知邻格count选方向"是否破宪法1? 读世界状态定参数(非手写谁强), 但"主动找空不浪费f"给隐性优势, 与固定方向基元不对称。**需用户确认是否夹私货**; 若算则退化为固定最近/随机2格。
- **风险2**: M-FB 需内部计数器(爆发/静息相), 动态状态非静态序列函数。实现上把"上次触发后经过步数"当外部时钟注入, 需 step loop 确认计数器挂哪。
- **风险3**: 多 F 堆叠时 p_leave 怎么复合(连乘 vs 取最大)? f 取复合还是各自独立触发? 须在基元复合规则统一拍。
- **风险4**: R-F4+R-F2 原地堆叠满格时全蒸发空转, 扩张全靠外力(对抗打空位); 分析侧须确认"被动扩张"不被误判为"无繁衍能力"干扰 φ。
- **风险5(跨池)**: M-FB 爆发依赖"恰在对手被打空后"触发→繁衍时钟与对抗触发有相位博弈; 但 step loop 若固定"对抗先于繁衍", M-FB 总在对手空出后才爆发, 节奏博弈空间被隐性压扁, 须 step loop 拍板时确认。

### 对抗池 v1 (策划 agent a23778)

**三岔路推荐:**
- 岔路1 → **同时结算(快照法)**: 用(a₀,b₀)双向算 kills_A→B=min(b₀,a₀·z_A) / kills_B→A=min(a₀,b₀·z_B) 再一起扣。否则先手免疫反击, z 结果依赖调度顺序。
- 岔路2 → **v0+明确分摊**: total_damage_to_X=Σkills_i→X, actual=min(b₀_X,total), 超额各攻击方按贡献比例分摊 actual_kills, 自损按实际击杀反算。"能打的抢到更多击杀"自然涌现。
- 岔路3 → **防御暂不做, 但防御从猎物清单自然涌现**: 某株身上没有任何攻击方猎物清单里的 motif → 攻击启动不了=等效完全免疫(最干净, 纯靠序列组成)。以后需要再加 D 基元(占插槽掩盖 motif)。

**对抗池基元清单 (v1):**

Residue 级:
- **Z-r1 Epitope Spike**: z=0.6(不划算), 瞄 F 类繁衍基元(任意F), 每3步。骚扰繁殖者。频率依赖=F多的株目标暴露宽被骚扰重/纯对抗株免疫。生物=抗体识别高表达抗原区。
- **Z-r2 Attrition Bite**: z=1.5(划算), 瞄 Z 类对抗基元(任意Z), 每6步。专克对抗株。频率依赖=Z多的战士型吸引围剿, 多数派战士被消耗, 稀有战士安全。生物=抗独特型抗体(anti-idiotype)。

Motif 级:
- **Z-m1 Predator Lock**: z=2.5(强), 瞄 Z-r2 motif, 每8步。战士杀手的杀手(克制链第三环)。频率依赖=Attrition Bite多时大量空名额扩张/稀少时无猎物消退。生物=构象特异性抗体。
- **Z-m2 Broad Sweep**: z=0.4(低,通才代价), 瞄3种(F-m+P-m+Z-r1), 每4步。宽谱弱攻无处不在。频率依赖=多样化群落综合伤害不低/单一组成被高z专才反杀。生物=天然多反应性抗体(polyreactive)。
- **Z-m3 Ambush Coil**: z=3.5(最高,折叠依赖需骨架特定位激活), 瞄 Z-m2 motif, 每12步(最慢)。专克宽扫型。频率依赖=Broad Sweep流行时爆发涌现/通才清场后退潮。生物=亲和力成熟高亲和抗体 vs 生殖系多反应。
- **Z-m4 Epitope Bleed**: z=1.0(等价交换), 瞄 Z-m3 折叠区特征段, 每5步。识别消耗伏击型令其无法积累。频率依赖=Ambush Coil膨胀时对等消耗锁平衡。生物=免疫逃逸株反识别原型抗体。

**克制结构(灵魂产出)**: 两条循环链。①Broad Sweep(通才)→被Ambush Coil(z3.5)克→被Epitope Bleed(z1)克→回环(Bleed猎物是Coil, Coil稀少时Bleed无收益退出; Sweep基数大时宽谱触达Bleed)。②Attrition Bite→被Predator Lock(z2.5)专克→Lock猎物(Bite)稀少时自损无收益退场→Bite重新崛起。**频率依赖三机制**: 多数派诅咒(任一类型主导→天敌猎物丰盛→天敌扩张→主导者被压→天敌失猎物萎缩→下一类型趁虚)、稀有者占优(正统频率依赖负选择, φ核心信号)、通才-专才权衡(Broad Sweep z0.4×3猎物 vs Predator Lock z2.5×1猎物, 生态位宽度与z内在权衡)。

**设计原则**: 高z(2.5,3.5)全配窄猎物(1种)防"高z+宽清单"无敌组合; 低z(0.4,0.6)才允许宽清单 → z量级与宽度强制反相关=最简无退化保障。折叠依赖只给最高z的Ambush Coil(折叠=天然扩散瓶颈)。Epitope Spike瞄F类制造对抗-繁殖跨池张力。

**对抗池待拍/风险:**
- Ambush Coil 折叠激活条件留白(等骨架数值轮); 须至少1始祖株骨架能支撑它否则永远沉默。
- Epitope Bleed z=1锁平衡可能变无聊静态; 建议 z=0.8~1.2 由骨架调制动态打破。
- Broad Sweep 3种猎物 motif 须在四始祖株分布不均(否则等效无差别攻击失去频率依赖)。
- Attrition Bite 专克Z类, 开局Z比例低则无猎物自损灭绝(正确频率依赖但让始祖株组成敏感); 建议至少2始祖株含Z基元。
- **跨池耦合(对抗池自报)**: ①Z-r1直读F类=对抗-繁衍唯一直连(繁殖快本身带对抗压力, 有意); ②Z-m2猎物含P-m=突变率高的株更易被宽谱捕食("突变代价": 突变非免费旋钮, 生物对应SHM高突变株CDR更易被识别); ③**折叠WHAT建议优先加 f 而非加 z**, 否则 Ambush Coil 折叠+z3.5叠加过高, 对抗池z分布无法独立管理。

### 跨池冲突 (hub 识别, 待用户拍)

- **冲突1: 折叠强化 WHAT — 两池给对立建议!** 突变池: 加 z + 加抗被猎, **明确不加 f**(折叠加繁衍会糊 f 来源, φ分不清f是F类还是折叠给)。对抗池: **优先加 f, 别加 z**(否则Coil折叠+z3.5叠加过高, 对抗池z分布无法独立管理)。→ 这是真冲突, 必须用户拍。

### ① 系统架构师 (agent a570e0): step loop

**最终顺序 = 快照驱动三阶段, 突变嵌入繁衍内部:**
- PHASE 0 取快照 world_snap = deepcopy(world)。本tick所有基元的"量"读快照, 不读活世界。
- PHASE 1 对抗(基于快照): 每格每对(甲攻乙, 猎物清单命中)且 T%Z_period==phase_offset → 用快照(a,b)算kills, 暂存delta, 批量写入world。
- PHASE 2 繁衍+突变(量读快照, 空位读对抗后活世界): 每株 T%F_period==offset → candidates=a_snap·f; 突变嵌入=落地时 candidates·p(x) 个带突变(分裂成 s 的 candidates·(1-p) + mutant(s) 的 candidates·p, 进 pending_arrivals); 迁出 world[g][s]-=a_snap·p_leave 写活世界。
- PHASE 3 K墙仲裁(消化 pending): total_arriving vs available=K-占用; 装得下全落, 装不下按 ratio=available/total 等比例分, 超额蒸发不挤活体。

**节奏博弈张力的裁决**: 繁衍池怕"对抗先跑→爆发型F总在对手被打空后才爆发"。架构师判定**顾虑部分成立但结论错**: 爆发型F的博弈空间是跨tick时序选择(何时撒撒多少), 不是同tick内顺序; 若改"繁衍先于对抗"反而给F单tick隐性先手(先占位破坏等比例仲裁=污染ground truth)。**快照保护**=繁衍读快照a, 对抗先不先跑只影响空位多少(K墙松紧)不影响F撒多少。结论: **对抗→繁衍(含突变)→K墙 保持**。

**时钟调度 = 异相位(个体出生记 birth_tick)**: 触发条件 (T-birth_tick)%period==0。同相位会全场每N步同步爆发(功率谱强峰, 非生物现实); 异相位平滑、接近真实克隆动力学。多基元独立时钟无冲突(各用各的 (T-birth_tick)%period)。

**对抗三岔路确认**: (1)双向同时结算→确认(快照双向算一起扣, 拒先后避免遍历顺序影响); (2)多株叠加v0→确认+补全(raw_damage>b_snap 时 ratio=cap/raw 按贡献比例分摊, 攻击方自损同比例缩, 双向都不出负count); (3)防御暂不做→确认("身上无攻击方猎物清单motif=等效免疫"已是涌现防御)。

**架构师标的耦合/风险**: A. PHASE 2 只生成pending绝不提前落地, PHASE 3统一仲裁(否则多源抢同格无法等比例); B. 迁出量取 min(a_snap·p_leave, a_snap) 防负; C. mutant(s)可能是全新序列, PHASE3落地往dict插新key, 收敛突变(变出已有序列)按序列作key自动合并; D. **基元table需加3字段: period(时钟周期)/phase_type(在哪PHASE执行)/read_from(读快照还是活世界)**; E. **折叠强化计算位置=PHASE 3仲裁内部、落地写入之前, 落地时一次性算(不是每tick维护状态)**。

### ③ 复合规则师 (agent a97b72): 基元复合

**同类堆叠快速对照表**: f → `1-Π(1-fᵢ)`(概率独立叠加, 任一F触发即有后代, 自然天花板<1); 波及格 → 并集各自独立撒(同格收 a·f₁+a·f₂ 受K墙限流); p_leave → `1-Π(1-p_leave,ᵢ)`(同f); z → `Σzᵢ`(交换比=多把刀真实相加, 暂无cap); 猎物清单 → `∪targetsᵢ`(广谱vs专一两条路); p(x) → `max(μ, 1-Π(1-pᵢ(x)))`(独立叠加+μ地板); spectrum q → `Σpᵢ(x)·qᵢ,ₑ / Σpᵢ(x)` 归一化(按各P触发率加权, 触发强的话语权高)。

**跨类协同 = 通道正交, 无跨类交互项**(F/Z/P各管各的, 互不相乘)。理由: 一旦隐式交互(如"Z大的株自动f小")=隐藏φ破宪法2; 混搭真实代价=有限插槽被瓜分(已内建不需额外惩罚)。**折叠组合技是唯一合法跨类耦合通道且显式声明**。

**计算管线顺序**: Step A 每基元独立原始输出 → Step B backbone乘性调制(通道特化窗口衰减, 夹回合法域) → Step C 折叠组合技 → Step D 同类复合。**裁决 backbone调制(B)先于同类复合(D)**(每基元是backbone语境下修正后零件, 用修正值参与复合, 保住"同基元插不同位不同效果"的分辨率)。折叠(C)在调制后复合前。

**折叠组合技formula形状 = 解锁新通道 + 组合系数α乘法(否决"单纯加成"语义与调制重叠/"非线性混合"特设难标定)**: Z+P折叠配对→新通道"攻击命中附带突变注入"(α·p_b概率施加spec_b编辑); Z+F→"猎后繁衍"(命中时胜者额外α·f_b拷贝); P+F→"分裂突变"(折叠撒出拷贝突变率×(1+α)); Z+Z→无组合技(z相加已处理)。抗被猎formula: 区内motif可见性=1-β_fold(对手清单以β概率遮蔽区内motif), 是折叠默认效果与组合技叠加存在。

**复合师标的风险**: 5.1 **z无cap可能成无脑最优**(z叠满→kills=min(b,a·z)恒触发乙全灭甲损b/z极小→零代价灭敌), 建议soft cap如log(1+z)或B类gain上限, formula先写线性相加cap留数值轮; 5.2 复合管线每tick缓存一次, 各基元自带时钟决定本tick激活否, 只有激活的参与复合(F4周期5+F8周期7仅在公倍数tick才都激活完整叠加); 5.3 Z+P组合技触发条件建议"Z触发时查P当前是否激活(各自时钟独立判断), 两者均激活才触发"; 5.5 折叠α乘在调制后量上→backbone布局与折叠位双重上位效应(有意, 但数值轮防爆表)。
