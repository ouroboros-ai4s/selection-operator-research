**a_invivo 机制小结(序列→可选表型链条):**
序列 → BCR 内禀键参数 (E_b 结合能垒, x_b 键长) → 力 f 下 off-rate k_b=k_0·e^{−β(E_b−f·x_b)} [Bell] → 与 Ag-tether 键 k_a 竞争断裂 (tug-of-war) → 提取概率 η / 提取量 n (=可选表型 a_invivo) → T 细胞按 presented antigen 量给生存信号 → clonal fitness。可选表型 ≈ 提取量 n,由 BCR-Ag 与 Ag-tether 两键**相对** off-rate(都含力 f)决定,非单边 K_D。

**a_invivo 五个承重让步(= 建模真缺口):**
1. **力 f 从哪来、是否序列函数未解**:P1/P3 把 f 当外加参数,讨论「noninheritable forces」。若 f 不可遗传/不由序列决定,则 a_invivo=a(序列, f_环境),序列单独不能定 a_invivo。**f=φ∘a 可辨识性关键耦合点**。
2. **BCR slip vs catch 未实测确认**:见上。
3. **K_D→提取量在高亲和区饱和(affinity ceiling)**:P2/P3/P9 三家指 tether 强度封顶,高亲和序列差不再转成提取量差。a_invivo 在高亲和端**信息坍缩**,a_DMS 高分辨被体内抹平。
4. **APC 类型(FDC 硬 vs DC 软)改变 a_invivo**(P6):a_invivo 非 well-defined 单函数,依赖选择发生在哪种突触/GC 阶段,与 σ 环境依赖纠缠。
5. **affinity-independent 区间边界未定**(P7/P8):体内何时按亲和排序、何时完全 permissive,机制(steric/antibody feedback/presentation 差异/tensional sensing)并列罗列未量化。
**结论**:a_invivo 解析候选已有(Bell-based tug-of-war 提取量),但显含不由序列决定的力学环境变量 f 与 APC 刚度,且高亲和端饱和 ⇒ a_DMS→a_invivo 既非单调也非可逆,坐实 B-S2,预示 a_invivo 可辨识性本身受限。

**σ 承重让步:**
1. σ_dead=0(死者不可采样)是硬假设非测得;GC 灭绝谱系永久缺席,投影严格多对一不可逆。

3. MTBD 似然两处错指定:① 要求 type 演化 Markov,而 GC type 由序列决定、序列突变非 Markov;② 无法容纳 carrying capacity ⇒ 虚假早 plateau(Ralph 明判为模型伪影非真 φ)。

**Louca-Pennell 迁移到 σ 的判断(子代理推理,非文献直述,已标注):**
- **可迁移**:GC BD 模型与宏演化 BD 同数学族;LP congruence-class 论证依赖「只观测存活者(extant)+似然只约束 dLTT」。σ 投影把 GC 数据降维到同类「存活者似然」⇒ **LP 式多对一不可辨在 σ∘φ 上结构性成立**(与 B-S1 φ 因果不可辨同源:不同 (φ,σ) 给同一观测树分布)。
- **两救赎杠杆**:(a) LP 不可辨仅对 extant-only/单时点成立;FBD 证「采样在过程中持续(removal prob r≠1)→时变 BD 重获可辨」,replay 时序 bulk+DMS 恰提供 LP 缺的那一维 ⇒ σ 可破投影理论依据;(b) piecewise BD 可辨(Legried-Terhorst),Morlon 区分 asymptotic/practical + 约束/正则化/外部数据可恢复可辨。
