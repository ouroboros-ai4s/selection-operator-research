### ★ 必须先纠正的框架误读(load-bearing)

用户的心智模型是"一套理论被打了个粉碎,一地碎屑"。**这个描述是错的,且这个错误会直接误导 round 3 的方向选择。** 诚实的事实:

1. **被打碎的只是装饰层,不是承重层。** 对抗轮 BROKEN 的是"四语言同构"和"单一骨架统一三层"——这两条都是 KAC 分类里的 CONVENIENT(美学驱动)。承重层(1 动力学原理的两条硬禁止 Price 协方差同号 / 噪声-N_e 标度)是 SUPPORTED,devil's-advocate 明确攻不动。
2. **真正的状态不是"碎屑",而是"一具干净、诚实、可证伪、但从未上过测试台的骨架"。** 核心预测 P1/P2 的档位是 **TESTABLE-NOT-YET-TESTED**——不是"被推翻",是"还没测"。
3. **所以"碎屑没法帮助实际科学"这句话,前提就不成立。** 不是碎屑;是一个待验的、结构清楚的假说。它现在帮不上实际科学,不是因为它碎了,而是因为它**还没被验证过**——而验证它恰恰就是"实际推动科学"本身。

### ★ 用户两个诉求的真实关系:不是 trade-off,是同一件事

用户要 ①"更深层结果" + ②"实际推动科学非发表"。表面像要平衡两个目标,实际上对抗轮的诊断已经证明:**在未测骨架上继续堆纯理论 = 既不更深也不推动科学,而是把碎屑堆得更高。** 理由:

- 对抗轮已诊断"过度声称"恰恰来自"在缺测试的情况下做理论"(elegance-trap + N_eff≈1.3 框架诱导收敛)。再做一轮纯理论而不引入判别第四路 = **重犯刚被诊断出的失败模式**。
- 一具理论唯一接触现实的接口是 P1/P2(可证伪预测)和 wet-lab 可行的最小介入交付物。不跑这些,"推动科学"无从谈起,"发论文"反而是纯理论唯一能干的事——正好是用户想避免的。
- 因此:**当前能拿到的"最深结果",不是更多理论,而是把 TESTABLE-NOT-YET-TESTED 变成 CORROBORATED 或 BROKEN。** 那才是真知识增量,且那个过程本身强迫你深化(定 θ/ε 阈值、造 5 个对抗真值世界、B5 子模性诊断——这些"更深的结果"只能由执行逼出来)。

**A — 执行轮(DIY-4 sandbox):把设计好但从未跑的测试跑出来。** 建个体级沙盒(带 7 条约束),跑 P1/P2、跑判别第四路、测 θ/ε 阈值、包络内外分测、B5 子模性诊断。

- 风险:工程量大(个体级引擎 + 5 真值世界 + 恢复方法 + 基线)。

### ★ 新增硬约束(用户明确要求,2026-06-13):真实开源数据基底
沙盒**不得**是"拍脑袋随便模拟出的异常简单数据";必须**在已有真实开源 dataset 基础上适度合成**。

**编排者必须先surface的中心张力**:要验证恢复方法必须有 ground-truth φ;真实数据恰恰没有 ground-truth φ(Weinstein 不可辨)。故纯真实数据沙盒逻辑上无法验证恢复方法。合成真值核心是**逻辑必需**,非偷懒。真实数据的作用 = 让合成核心真实 + 提供独立现实接触。

**混合架构(角色分配,落进 brief):**
- **真实数据 3 个承重角色**:① a_DMS 亲和地形 = 真(Phillips 2021 CR9114 2¹⁶=65536 / CR6261 实测,本地已有 parquet,带真实 epistasis);② 突变算子 M = 真(S5F/SHM 已发表模型);③ 种群尺度 N/μ/轮次 = 校准到真实 repertoire 数据量级。
- **合成严格只限"待恢复的三算子"** φ/σ/a_invivo,且按非循环铁律② **不手写真值函数**:写个体级机制规则(Bell-Evans 力依赖 off-rate / 硬阈值 σ / 带地板天花板竞争的 φ)→ 跑个体级 → 真值**涌现并记录** → 恢复方法只见幸存者序列时序,永不见规则。
- **保真度校准(让"适度"可检验非拍脑袋)**:任一合成真值世界启用前,先让模拟统计量(克隆大小分布/SHM 谱/多样性衰减/富集比分布)对真实 repertoire 数据;对不上的世界不许用。

**两个验证 regime:**
- **Regime 1 合成真值(方法验证)**:5 个对抗真值世界 + P1/P2 + 命题① θ/ε + B5 子模性诊断都在此(需已知真答案)。成功标准=方法诚实报告失败,非硬套假成功。
- **Regime 2 真实数据(撞现实,无真答案)**:① P1 上真实克隆干涉区时序(若可得);② 判别第四路=真实 CR9114/CR6261 DMS 上对测度流盲的纯统计自底向上拟合(收敛=独立佐证,直接回应 N_eff≈1.3);③ 包络检查=真实成熟是否落在适用包络内。
