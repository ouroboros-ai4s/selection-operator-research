## Checkpoint 靶A-同构: isomorphism-falsification 攻 A4 四语言同构

### 被攻主张(A4,头号嫌疑)

R_irr 这一个对象在四种数学语言里"是同一个东西",且都=Fisher 算子 Φ*Φ 的核:① 因果 hedge(Shpitser-Pearl 不可辨 hedge 结构)= ② 反问题零奇异向量(Φ 的零空间)= ③ 场论 Doi-Peliti 零模 = ④ OT 非保守(curl)分量。"同构"是精确数学词,burden 极重。攻击目标:找出这主张能站住的最高档位(rung 1-5),据此校正措辞。

### Step 4 先做(最便宜的一刀):维数/计数 sanity

地图声称 R_irr 由**两个生成元**张成(N_e·s 选择强度标度对称 ⊕ s−κ 的 σ-φ 移位对称)。同构的必要条件:四种语言各自**独立**算出的"零集"维数都=2(或至少一致)。逐语言数:

- **② 反问题零奇异向量**:dim = Φ 的零奇异值个数。这是定义良好、可数的——给定 Φ 的离散化,数 σ_i=0 的个数。地图主张=2。**可独立检验**(sandbox 里算 Φ*Φ 谱)。
- **④ OT 非保守 curl 分量**:dim = drift 场 Hodge 分解里非保守(旋量)部分的维数。在序列空间(高维离散)上,curl 分量的维数一般**远大于 2**——Hodge 分解的非保守部分维数 = 一阶上同调群维数,由空间拓扑定,通常 ≫2。**这是第一个危险信号**:除非序列空间拓扑特别简单(单连通使 H¹=0 则无 curl,或特定结构使 dim=2),否则 ④ 的维数与 ②的"2"对不上。
- **① 因果 hedge**:hedge 是图结构对象(C-forest/hedge 结构),它的"维数"不是自然的向量空间维数,而是**不可辨参数的自由度**。把它强行赋一个维数本身就需要论证。地图主张它=2(两对称)。**可疑**:hedge 的自由度由因果图的结构决定,凭什么恰好=2?
- **③ 场论零模**:Doi-Peliti 作用量在鞍点附近的零模(Goldstone 模),数=被自发破缺的连续对称性的生成元数。若真有且仅有两条连续对称(标度+移位),则零模数=2。**这一支自洽**——但前提是"且仅有两条"未被证伪(这正是靶B 命题②要查的)。

**维数 sanity 初判**:②和③可能给 2(②须 sandbox 验,③依赖"仅两对称"未被证伪);**④ curl 维数高度可疑,一般 ≫2**;①hedge 维数概念本身需先论证。四语言**不是**显然都给 2 参数零集。**A4 在维数计数这关已经出现不一致风险**——尤其 ④ OT curl。若 sandbox 算出 curl 维 ≫ 2,A4 当场跌到 rung 4(共享不变量)以下,因为连"同维"都不成立。这是最便宜、最该优先在下一轮 sandbox 做的一刀。

### Step 1+2 钉死四对象 + 构造两对承重 map

**钉死四对象的精确身份**(SKILL step1:写不清=本身是 finding):
- ② 反问题零奇异向量:ambient=序列上的参数扰动空间;零集={v:Φv=0};结构=向量空间(Φ 线性近似下)。**定义最干净**。
- ① 因果 hedge:ambient=因果图上的不可辨分布族;"零"=do 表达式不被观测分布唯一确定的方向;结构=图论对象(hedge=两个 C-forest 的特定配置),**非自然向量空间**。
- ③ 场论零模:ambient=Doi-Peliti 作用量的场配置空间;零模=作用量二阶变分 Hessian 的零本征方向(Goldstone);结构=切空间向量。
- ④ OT 非保守 curl:ambient=drift 向量场空间;curl=Hodge 分解的非保守分量;结构=向量空间(一阶上同调)。

**承重 map A(最可查):② 反问题零奇异向量 ↔ Fisher Φ*Φ 核。** 这对几乎是定义性的:Φ*Φ 的核 = {v:Φ*Φv=0} = {v:‖Φv‖²=0} = {v:Φv=0} = Φ 的零空间 = 零奇异向量张成的空间。**这对 map 成立,且是 rung 1(线性代数恒等式)。** 但注意:这只是说"Fisher 核 = 反问题零空间",这俩本来就是同一对象的两个名字,**不构成四语言同构的实质内容**——它是同构里最 trivial 的一环。真正的 burden 在跨语言的另几对。

**承重 map B(最大胆):① 因果 hedge ↔ ③ 场论零模。** 要写一个 structure-preserving 双射:每个 hedge ↔ 每个 Goldstone 零模。**写不出来**:hedge 是离散图论对象(有限个 C-forest 配置),零模是连续切空间的方向。一个离散集合与一个连续向量空间之间不存在自然双射(基数都不同,除非 hedge 退化到恰好参数化一个连续族)。**这对只能 gesture("它们都表示不可辨方向"),写不出显式 map → 按 SKILL step2 落 rung 4。**

### Step 3 monster-hunting

- **Monster 1(打 map B 的双射性)**:catch-bond 非单调情形——存在一个不可辨方向,在场论里是零模(连续形变作用量不变),但在因果图里**不**对应单个 hedge(它跨越多个 C-forest 配置)。一个零模对多个 hedge → 非单射 → map B 即使勉强写出也不是 iso。
- **Monster 2(打 ④ curl 的归属)**:序列空间若非单连通(突变图有环,真实情形如此),Hodge 分解的 curl 分量维数=H¹ 维 ≫ 2。这些额外 curl 方向**不**对应任何 N_e·s 或 s−κ 对称 → ④ 的零集含"杂项",不与①②③的 2 维零集等同。**这个 monster 同时打 A4 和靶B 命题②(存在不由 gauge 诱导的 hedge)。**
- **Monster 3(打①的向量空间结构)**:hedge 集合无加法结构(两个 hedge 之"和"无意义),但②③④都是向量空间(零向量可线性组合)。结构不匹配 → ① 与 ②③④ 之间不可能是保结构同构,最多是集合层面的对应。

三个 monster 都活着,无一被 monster-barring 排除(排除它们需把 R_irr 收窄到"单连通序列空间+单势垒+线性 Φ"的特例,那就不是原主张了)。

### Step 5 定档 + 桶判定

**A4 最高可辩护档位 = rung 4(共享不变量),局部可达 rung 2。** 具体:
- ② ↔ Fisher 核:rung 1(恒等式,但 trivial)。
- 线性化 + 单连通 + 单势垒特例下,②③可能 rung 2(在零模子空间上同构)。
- ① hedge 与 ④ curl 卷入后:跌到 **rung 4**——四者"共享一个不变量"(都标记某种不可辨自由度,且在最干净特例下维数可凑到 2),但**无四向保结构双射**(①非向量空间;④维数一般≫2)。
- 全局(非线性、多连通序列空间、catch-bond):**rung 5(类比)**。

**桶判定:A4 = BROKEN(作为"同构"主张)。** 但这是**建设性 BROKEN**,非全盘否定:真实成立的是 rung 4 的"共享不变量 + 特例 rung 2 子结构同构"。**须改的措辞**:全景地图所有写"四语言同构(isomorphism)"处,降级为——"四语言在**线性化、单连通、单势垒特例**下共享一个 2 维不变量,其中②反问题零空间与③场论零模在该特例下子结构同构(rung 2);①因果 hedge 与 ④OT curl 与之只是**结构类比**(rung 4-5),不构成保结构双射。" 这反而是更强的真陈述(它精确说出在哪同构、在哪只是类比),符合 SKILL 第42行"rung 2 是 publishable-grade 真陈述,假称 rung 1 是自欺"。

**可证伪性**:A4 高度可证伪(维数计数+map 构造都是硬数学),且已被证伪(降档)。**非 UNFALSIFIABLE**——这是好事:它说了硬话,所以能被检验,检验结果是"过度声称"。

**下一轮 sandbox 验证判据(便宜优先)**:算 Φ*Φ 谱核维 + drift 场 Hodge 分解的 curl 维。若 curl 维 ≫ 核维,确认 ④ 不属同一 2 维不变量,rung 4 判定坐实。这是最便宜一刀。

## Checkpoint 靶A-骨架: elegance-trap-probe 攻 A1/A2(直接服务用户奥卡姆顾虑)

### 被攻主张

A1(不存在 per-seq 静态 φ)+ A2(存在单一测度流 μ_t,f=σ∘φ∘a 是三投影)。核心问题(用户的怕):骨架的优雅是 EARNED(靠禁止+预测挣来的简约,如自然选择/scaling law)还是 DECORATIVE(漂亮的再描述,如弗洛伊德框架解释一切却禁止什么都不禁止)?

### Step 1 禁止集构造(骨架禁止什么可检验的观测?)

逐条逼问"什么 affinity-maturation 数据模式在骨架为真时是**不可能**的":

- **禁止 1(真禁止)**:骨架(测度流=Fleming-Viot 生成元,选择项=Cov_μ(φ,s))**禁止"选择响应方向与群体内 φ-s 协方差符号相反"**。Price/Robertson 类关系:Δ平均性状 与 Cov(适应度,性状) 同号。若 sandbox 观测到群体朝 Cov 为负的方向稳定演化(选择把低协方差性状推上去),骨架破。**这是可检验的硬禁止。**
- **禁止 2(真禁止)**:骨架把 X1 噪声律**导出**为有限群体重采样项,故**禁止"噪声幅度与有效群体大小 N_e 无 1/N_e 标度关系"**。若 sandbox 测得噪声幅度独立于 N_e(或反向标度),骨架的"噪声=重采样导出"破。**可检验。**
- **禁止 3(弱禁止/边界)**:骨架主张 f=σ∘φ∘a 三投影,**禁止"存在一层无法表示为该过程投影的独立自由度"**——但这条**危险地接近不可证伪**:因为"投影"很灵活,几乎任何观测都能被宣称为"某个投影"。这正是 SKILL 第17行警告的 decorative 风险点。**这条须靠靶A 同构已暴露的硬数学(curl 维≫2)来钉**:若 ④ 有不属 2 维不变量的额外自由度,那就是"无法被 2 生成元过程吸收的独立自由度",禁止 3 被自身框架内的对象违反。

**禁止集初判**:骨架**不是空的**——禁止 1、2 是具体可检验的硬禁止(都落在群体遗传学的 Price/漂变标度上)。这把 A1/A2 从"纯 DECORATIVE/UNFALSIFIABLE"里救出来了。但禁止 3 暴露:骨架的"三投影"表述有一个 decorative 软肋(投影太灵活),须靠同构攻击的硬维数结果约束。

### Step 2 风险预测猎取(与旧 φ∘a 图像不同、sandbox 可检的预测)

- **风险预测 P1**:旧 φ∘a(静态适应度)预测"终点富集比单调反映 φ";骨架预测"在克隆干涉(Nμ≫1)区,终点富集比与 φ **非单调**(测度流的非保守分量使富集偏离 φ 梯度)"。**sandbox 可检**:扫 Nμ,测富集比-φ 关系是否在高 Nμ 区偏离单调。**这是骨架挣 EARNED 的关键预测**——它说了旧图像不说的话。
- **风险预测 P2**:骨架预测"do(冻结环境 e_t≡const)会使非保守 drift 分量塌缩、富集比恢复单调反映 φ";旧图像无此预测(它没有"非保守分量"概念)。**sandbox 可检**(冻结环境介入)。

P1/P2 都存在且与旧图像可区分 → 骨架**有风险预测**,不是纯 accommodation。

### Step 3 accommodation 审计(三通道是被骨架预测,还是事后塞进去?)

诚实查历史:我们的实际研究路径是**先猎到 18 卡 → 再找统一子**(v3 相2)。所以先验怀疑很高:骨架可能是 fitted around 三通道、让它们看起来统一,而非预测了它们。逐通道判:

- **通道1(测度值流)**:**predicted-by**。一旦接受"群体级测度流"骨架,Fleming-Viot 生成元的三项(突变+选择漂移+重采样)是数学**推论**,不是事后塞的。通道1 是骨架的自然产出。✓ 真预测。
- **通道2(σ 力学读出)**:**accommodated-into**。σ 的力学可观测化(n_Ag 拉力提取)是从**生物物理**独立来的,骨架并没预测"必须测力"。相2 自己也承认"σ 不能被动力学统一子吸收"——这恰恰说明通道2 是**塞进去的**,不是测度流骨架生出来的。✗ 事后整合。
- **通道3(部分辨识+最小介入)**:**partially predicted**。一旦把问题写成算子 Φ 反演,部分辨识(Manski bounds)是反问题语言的自然推论;但"最小介入"是因果推断独立带来的。半预测半整合。

**accommodation 审计结论(关键诚实):骨架真正生成的只有通道1。通道2 是独立的生物物理,通道3 半独立。** 这与靶A 同构的发现一致(σ/hedge 那几支不属同一对象)。**这说明"单一骨架统一三通道"被高估了——更诚实的是"一个动力学骨架(通道1)+ 两个不被它生成的独立支柱(通道2/3)"。** 这正是用户怕的反面验证:不是"一个原理解释一切",而是"一个原理 + 两根它管不到的支柱"。

### Step 4 删除测试(删骨架留三通道,丢预测力还是只丢故事?)

设想删掉"单一测度流骨架"的叙事,只保留三通道作独立方法:
- 通道1 仍能独立跑(FV 拟合 + Hodge 分解)——但**会丢失** P1/P2 这两个风险预测(它们来自骨架把富集比与非保守分量联系起来)。**丢了预测力。**
- 通道2/3 作独立方法**几乎无损**——它们本来就不靠骨架生成。
**删除测试结论**:删骨架对通道1 **丢预测力**(P1/P2 没了),对通道2/3 只丢叙事整齐。所以骨架对通道1 是 EARNED,对通道2/3 是 DECORATIVE(组织框架)。

### Step 5 定档(骨架的 EARNED/DECORATIVE 判定)

**A1/A2 桶判定:PROMISING-BUT-UNEARNED,且须拆分表述。** 精确结论:
- 骨架对**通道1(动力学层)= EARNED**:有硬禁止(禁止1/2)、有风险预测(P1/P2)、删除丢预测力。这部分是用户要的真优雅。
- 骨架对**通道2/3 的"统一"= DECORATIVE**:是组织框架而非发现的定律,删除只丢叙事。**须降级措辞**:不说"单一测度流统一了 σ∘φ∘a 三层",改说"动力学层由测度流骨架 EARNED 地生成;σ 力学层与因果辨识层是与之**正交的独立支柱**,骨架为它们提供坐标系(组织框架)而非生成它们"。
- **命名预测 P(挣 EARNED 的路)**:P1(克隆干涉区富集比-φ 非单调)是骨架挣全局 EARNED 的关键——若 sandbox 验 P1 成立,骨架的动力学部分坐实 EARNED;σ/辨识两支柱则靠各自的硬禁止(σ 不可吸收须 do(n_Ag) 验)独立挣分。

**回应用户奥卡姆顾虑(本轮最核心的一句)**:用户怕"满是补丁"。靶A 的诚实结论是——**这不是 18 个补丁,但也不是 1 个原理解释一切;它是"1 个 EARNED 的动力学原理(通道1)+ 2 根它管不到、各自有硬禁止的正交支柱(σ/辨识)+ 1 个诚实残余(R_irr)"**。这个结构本身是简约的(3 根而非 18 根),且每根都有可证伪的硬禁止——符合奥卡姆,但**比原全景地图宣称的"单一骨架"诚实**:原表述把 2 根独立支柱也算进"统一",是 elegance-trap。降级后反而站得更稳。

**可证伪性**:A1/A2 可证伪(禁止1/2 + P1/P2 都硬)。非 UNFALSIFIABLE。初步桶=CORROBORATED(动力学部分)+ 须降级措辞(统一性部分)。

## Checkpoint 靶B-三命题: adversarial-stress-testing(Lakatos/Popper)+ counterfactual-probing(Pearl PNS/PS)

### 命题① 三塌缩子空间近似直和 + R_irr 小

**Lakatos monster-barring(找反例)**:靶A 同构攻击已经产了一个直接打①的 monster——若序列空间多连通,OT curl 分量维数 ≫ 2,且通道2(σ 力学)被 accommodation 审计判为"独立支柱不被骨架生成"。**这两点合起来强烈暗示三子空间并非干净直和**:σ 通道若真独立于动力学通道,它们可能正交(利好①的直和)也可能强共线(打①)——取决于 σ 与频率依赖是否纠缠。相2 发散已记 X5(σ-φ 力学纠缠),catch-bond 非单调正是 σ 与 φ 纠缠的机制。**若 X5 成立(σ 与 φ 纠缠),通道1 与通道2 子空间共线,①的"近似直和"破。**

**counterfactual-probing(PNS/PS)**:命题①其实是两个子主张——(a)三子空间近似直和;(b)R_irr 小。查它们的地位:
- (a)直和:**是承重假设,非已证结论**。PN(必要性):若非直和,四组正交分工的合法性崩(组I/II/III 互相污染)→ 直和对"四组分工"高度必要。PS(充分性):直和成立不保证 R_irr 小(可以直和但每个子空间都病态)。
- (b)R_irr 小:**纯希望,无论证**。地图从未证明 R_irr 维数相对全谱小,只是"希望"它小。**这是命题①最脆弱处**:R_irr 大是完全可能的(Weinstein 不可辨可能很大)。

**命题① 桶判定:UNFALSIFIABLE-LEANING → 须降级为"开放经验问题"。** 理由:"近似直和""R_irr 小"里的"近似""小"无定量阈值——多近算近?多小算小?没说阈值=不可证伪。**须改措辞**:把①从"主张"降为"待测量"——"三子空间夹角与 R_irr/全谱比值是 sandbox 待测量,当前无理由断言直和或 R_irr 小;命题①的乐观版(直和+小)是**有待验证的最好情形假设**,非已确立结论。" **证伪条件(下轮 sandbox)**:测 T_E 三子空间两两主夹角(<某阈值=近似直和)+ 测 dim R_irr / dim 全谱(<某阈值=小)。须先定阈值否则不可证伪。**这条直接服务奥卡姆诚实:不许把"希望它简约"伪装成"它简约"。**

### 命题② 全部 hedge 由 gauge 对称(N_e·s 标度 ⊕ s−κ 移位)诱导

**Lakatos monster-barring**:靶A 同构的 Monster 2 直接落在这里——若序列空间多连通,OT curl 分量维数 ≫ 2,这些额外 curl 方向**不**对应 N_e·s 或 s−κ 任一对称。**这就是一个不由 gauge 诱导的 hedge 反例。** 命题② 面临选择:要么 monster-barring(把 R_irr 收窄到单连通特例排除这些 curl),要么承认存在杂项 hedge。**收窄=承认命题② 只在单连通特例成立(monster-barring 是合法的 Lakatos 动作,但须明说收窄了定义域)。**

**counterfactual-probing(PNS/PS)**:命题② 的地位 = A5(R_irr=Noether 轨道)的另一面。PN:若存在非 gauge hedge,则"不可辨=对称性"的优雅论证破(R_irr 不再是干净对称轨道)→ 命题② 对 A5 高度必要。PS:命题② 成立(全 hedge 由 gauge 诱导)是否充分推出 R_irr 由两生成元张成?**不充分**——可能有第三、第四条对称我们没找到。地图只数了两条(标度+移位),凭什么穷尽?**这是命题② 的隐藏漏洞:它假设了"仅两条对称"却未证明无第三条。**

**命题② 桶判定:BROKEN(全局版)/ CORROBORATED(单连通特例版,但须明说收窄)。** 与靶A 同构判定一致。**须改措辞**:"在单连通序列空间 + 线性化特例下,hedge 由两条 gauge 对称(N_e·s 标度 ⊕ s−κ 移位)诱导;一般情形(多连通、catch-bond 非单调)下存在不由这两对称诱导的额外不可辨方向(OT curl 杂项),命题② 在那里不成立,且不排除存在未发现的第三条对称。" **证伪条件(下轮 sandbox)**:在多连通序列空间上算 drift Hodge 分解,检 curl 分量是否全部落在两 gauge 对称张成的子空间内(全落=②成立;有漏出=②破)。**可证伪,非 UNFALSIFIABLE。**

**靶B ①② 交叉发现(重要)**:命题①(直和+R_irr 小)和命题②(hedge 全由 gauge)都在"单连通/线性化特例"成立、在一般情形破或不可证伪。**这说明骨架的优雅高度依赖一组未明说的特例假设(单连通序列空间 + 线性化 Φ + 单势垒)。** 这本身不是错,但**必须显式写出这些特例假设**——否则"优雅统一"是在偷偷限定域上成立、却以普适面目呈现,这是比补丁更隐蔽的 elegance-trap。诚实做法:把这组特例假设提为骨架的"适用包络(validity envelope)",明说"在此包络内 EARNED 简约,包络外退化"。

### 命题③ φ 经 ΔG 因子化成立

**Lakatos monster-barring**:直接的 monster = **catch-bond**。catch-bond 的 off-rate 随力**非单调**(先降后升),这意味着 a_invivo 的力依赖提取不能只由平衡结合自由能 ΔG 决定——它需要一个**动力学**参数(势垒宽度 x‡、甚至多势垒)独立于平衡 ΔG。**若 φ 经 a_invivo 依赖于这个独立 on-rate/势垒结构,则"φ 仅经 ΔG 因子化"破**:存在 ΔG 之外的序列特征(决定 x‡ 的残基)直接进 φ。catch-bond 是真实抗体-抗原界面已观测的现象(力增强结合),不是边角反例——**这个 monster 强**。

**counterfactual-probing(PNS/PS)**:命题③ 的承重作用 = 支撑 T-Crooks(Crooks/Jarzynski 连 a_DMS↔a_invivo,gap=耗散功)。PN:若 φ 不经 ΔG 因子化,Crooks 桥的"gap=耗散功"解释失去单一 ΔG 锚点 → 命题③ 对 T-Crooks 高度必要。PS:命题③ 成立是否充分让 a_DMS↔a_invivo 可连?**不充分**——还需 σ 的力学读出。所以命题③ 是 T-Crooks 的必要不充分条件。

**命题③ 桶判定:BROKEN-LEANING(被 catch-bond 反例威胁),但可降级为"分层成立"。** 诚实结论:φ 经 ΔG 因子化在**平衡主导区(力小、单势垒、无 catch-bond)**成立;在**力依赖区(catch-bond、多势垒)**须额外 on-rate/势垒参数,ΔG 单因子化不够。**须改措辞**:"φ 在平衡主导区经 ΔG 因子化;在力依赖提取区(a_invivo 的 catch-bond 机制)φ 还依赖独立的动力学势垒结构(x‡),非 ΔG 单因子。" **证伪条件(下轮 sandbox)**:造 catch-bond a_invivo 真值,检 φ 恢复是否需 ΔG 之外的 x‡ 参数(需=③在力依赖区破)。**与相1 非循环矩阵呼应**:Bell 模型自带 ΔG 结构,手写它=种入③,故③须用"涌现的 catch-bond"非"手写 Bell"来非循环测试。

### 靶B 三命题总结

三命题全部**不是干净成立,而是"分层/特例成立 + 包络外退化"**:
- ① 直和+R_irr 小:UNFALSIFIABLE-LEANING(无阈值)→ 降为待测量,须先定阈值。
- ② hedge 全由 gauge:单连通特例 CORROBORATED / 一般情形 BROKEN(curl 杂项)。
- ③ φ 经 ΔG:平衡区成立 / 力依赖区 BROKEN(catch-bond)。
**共同模式**:三命题的优雅都依赖一组特例假设(单连通/线性化/单势垒/平衡主导)。**这是靶B 对全局最重要的发现——骨架须配一个显式"适用包络",包络内简约 EARNED,包络外诚实退化。** 这不削弱结果,反而让它从"宣称普适的漂亮故事"变成"明确划定适用域的真陈述"——正是奥卡姆诚实的要求。
