疑问1:16 是突变位点数,为什么字段叫 genotype 而不是突变 one-hot?

因为它不是 one-hot,是 multi-hot 二进制向量,叫 genotype 更准确。

- one-hot = 一个向量里只有一个 1(用来表示"N 选 1 的类别",如 [0,0,1,0] 表示"第3
- 我们这个 16 位向量里 可以有任意多个 1 同时为 1(如 1011000... 表示"第1、3、4 号突变同时发生")。这叫 multi-hot。

"genotype(基因型)"是生物学的准确叫法:它描述的是"这个体的 16 个突变位点各自处于立的二元开关:0=还是祖先的氨基酸,1=已变成成熟态的氨基酸。16 个开关 → 2^16 种genotype。所以它语义上是"16 个并行的二分类",不是"一个 16 选 1"。用 one-hot 这个词会误导。

疑问2:enum{H1, H3, B} 里 H1、H3、B 是什么?

是三种流感病毒的血凝素(hemagglutinin, HA)抗原——也就是抗体去结合的三个不同靶标:

┌──────┬──────────────────────────────────────┬────────────────────────────────┐

├──────┼──────────────────────────────────────┼────────────────────────────────┤
│ H1   │ Influenza A subtype H1 hemagglutinin │ 甲型流感 H1 亚型的 HA(如 H1N1)
├──────┼──────────────────────────────────────┼────────────────────────────────┤
│ H3   │ Influenza A subtype H3 hemagglutinin │ 甲型流感 H3 亚型的 HA(如 H3N2) │
├──────┼──────────────────────────────────────┼───────────────────────────────
│ B    │ Influenza B hemagglutinin (fluB)     │ 乙型流感的 HA                  │
└──────┴──────────────────────────────────────┴────────────────────────────────┘

CR9114 是广谱抗体,能同时结合这三种(因为它认保守的 HA 茎部)。同一个抗体变体对这三个抗原的结合力不同 —— 这个"换抗原结合力就变"的差异,就是条件依赖 c 的物理来源,也是我们要验证 f
学到的核心性质(I3)。文件里对应 h1_mean / h3_mean / fluB_mean 三组列。

疑问3:生物学重复次数是什么?minus_log_KD_replicates 是什么?

生物学重复(biological replicate) = 把同一个测量,从头独立地重做几次,看结果稳不 replicates=3)。

- 为什么要重复:高通量实验有随机噪声(细胞分选、测序波动)。测 1 次可能不准,测 3 次能估计"这个数到底多可信"。
- minus_log_KD_replicates = 这 3 次各自得到的 −log(KD) 值,即文件里的 h1_repa /
- 实测例子:某变体对 H1 的三次 = [9.5613, 9.5537, 9.4677] —— 三次很接近,说明这个测量可信。

疑问4:minus_log_KD_mean 就是这三次的平均吗?怎么算的?

是,就是算术平均,公式完全是你写的那个,我已在全表 65536 行核验过,零偏差:

$$\text{minus_log_KD_mean} = \frac{\text{rep}_a + \text{rep}_b + \text{rep}_c}{3}$$

例子:(9.5613 + 9.5537 + 9.4677) / 3 = 9.5276,与文件里 h1_mean=9.5276 完全一致。这个 mean 就是我们训练 f̂ 的回归真值 y。

疑问5:minus_log_KD_standard_error 是什么?英文全称?

英文全称:Standard Error of the Mean(均值标准误),缩写 SEM。

含义:它衡量"这个均值有多可靠"——值越小,说明这个 −logKD 测得越准。注意:它不是简  验过,文件里的值(如 0.0212)对不上 标准差/√3(算出来是 0.030 或0.0245)。说明它来自更底层的 Tite-Seq 测量模型(每次重复内部本就有上万个细胞的读数,SEM 是从那个拟合的不确定度传上来的),不是对 3 个 mean 取的。

它在我们方案里的作用:作为 Tobit loss 里的 σ(噪声尺度),把"测量噪声"和"模型拟合误差"分开。spec 里取的代表值 σ ≈ 0.047 就是这个量的典型量级。
