# 方案：写不出公式，那就「用模拟来推断」

## 反着来

$\varphi$ 写不出解析式，那就**不去写它，改成造一个真值已知的假世界，再检验「从数据反推 $\varphi$」这套方法行不行**。这套思路叫**模拟推断**（simulation-based inference，简称 SBI——用模拟器代替写不出的似然函数来做推断）。

四步：

1. **搭一个生发中心模拟器（引擎）**，让一堆虚拟 B 细胞在里面竞争、突变、被选择、死掉。
2. 引擎里的 $\varphi$（连同 $a$、$\sigma$）不是事后给结果贴的标签，而是**每一步真正决定「谁繁衍、谁死」的那只手**——既然是我们亲手设的，它的真值我们一清二楚，旁路记录下来当答案钥匙。
3. 正向跑出大量「序列 → 命运」的合成样本。
4. 用一个恢复模型从这些样本**反推 $\varphi$**，再跟偷偷记下的真值一对：对得上 = 这套推断方法靠谱；对不上 = 方法被证伪。
